#  Análise de Notas com Python

## Descrição
Projeto desenvolvido em Python com o objetivo de coletar, validar e analisar notas de alunos, gerando um relatório com métricas importantes.

## Funcionalidades
- Entrada de múltiplas notas
- Validação de valores (0 a 10)
- Cálculo da média
- Identificação da maior e menor nota
- Contagem de aprovados e reprovados
- Classificação de desempenho

## Tecnologias
- Python

## Objetivo
Praticar lógica de programação e conceitos iniciais de análise de dados.

## Como executar
1. Execute o arquivo `.py`
2. Insira as notas conforme solicitado
